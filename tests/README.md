# Tests

Test files for MyCareClaim (HealthcareAgentic).

## Browser Tests

Located in `browser/`. These are browser console scripts for manual testing.

**Usage:**
1. Deploy to Firebase: `npm run deploy`
2. Open browser DevTools console
3. Paste and execute test script

**Test Files:**
- `test_comprehensive_fix.js` - Comprehensive functionality tests
- `test_payment_flow.js` - Stripe payment flow tests
- `test_subscription_fix.js` - Subscription management tests

## Note

No automated test framework is currently in place. These scripts are designed
for manual browser console execution.
